# Screenshots Directory

This directory should contain 4 screenshots for the README:

1. `home.png` - 首页截图
2. `workspace.png` - 工作台截图
3. `streaming.png` - 流式输出中截图
4. `audit.png` - 审计报告截图

## Screenshot Requirements

- Resolution: 1920x1080 or higher
- Theme: Dark mode
- Format: PNG
- Take screenshots in the following order:
  1. Home page (landing page with hero section)
  2. Workspace with novel selected and pipeline idle
  3. Workspace during streaming output (generating chapter)
  4. Audit report panel with scores displayed

## Naming Convention

Use the format: `{description}_{width}x{height}.png`

Example: `workspace_1920x1080.png`
